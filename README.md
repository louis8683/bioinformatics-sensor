# Bioinformatics Sensor Project

## Introduction
This repository contains the code for the sensor of the bioinformatics project for high school students.

## Sensors
### PM2.5 sensor: PMS7003
### CO sensor: ZE07-CO
### Temperature and humidity sensor: DHT20
